## Customer Shopping Behavior Analysis

## 📌 Overview
This project analyzes customer shopping behavior using **Python**, **SQL**, and **Power BI**.  
It aims to uncover purchasing patterns, KPIs, and actionable insights that help businesses improve decision-making and customer engagement.

## 🛠️ Tools & Technologies
- **Python**: Data preprocessing, cleaning, and exploratory analysis (Pandas, NumPy, Matplotlib, Seaborn).
- **SQL**: Structured queries for customer segmentation and trend analysis.
- **Power BI**: Interactive dashboards for KPI tracking and visualization.
- **Excel/CSV**: Raw data handling and validation.

## 📂 Project Files
- `Business Problem Document.pdf` – Problem statement and objectives.
- `customer_shopping_behavior.csv` – Dataset.
- `Customer_Shopping_Behavior_Analysis.ipynb` – Python notebook for preprocessing and analysis.
- `customer_behavior_sql_queries.sql` – SQL queries for insights.
- `customer_behavior_dashboard.pbix` – Power BI dashboard.
- `Customer-Shopping-Behavior-Analysis.pptx` – Presentation slides.
- `Customer Shopping Behavior Analysis.pdf` – Final report.

## 📊 Key Insights
- Identified top customer segments based on purchase frequency and spending.
- Highlighted seasonal trends and product category preferences.
- Built KPI dashboards for revenue, customer retention, and anomalies.

## 🚀 How to Run
1. Clone the repository.
2. Open the Jupyter Notebook (`.ipynb`) for Python analysis.
3. Run SQL queries on your preferred database system.
4. Open the `.pbix` file in Power BI Desktop to explore dashboards.

## 🏆 Outcomes
- Improved understanding of customer shopping behavior.
- Actionable insights for marketing and sales teams.
- Scalable workflow combining Python, SQL, and Power BI.
